# 怎样才能连接到网络

## 通过路由器连接到网络

1. 将路由器的电源线接通。
2. 连接路由器入网接口与右侧WAN口, 连接电脑主机后的网线接口与路由器左侧LAN口。
3. 打开电脑中的浏览器, 输入192.168.1.1，进入路由器设置界面。第一次进入路由器设置界面需要自行设置密码。
4. 进入路由器设置界面设置路由器参数，参数设置正确后路由器即可连接到网络。
5. 路由器连接到网络后, 可以使用手机、笔记本电脑等连接无线网络，其中无线网密码为第四步中设置的密码。
