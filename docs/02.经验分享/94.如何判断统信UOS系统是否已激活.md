# 如何判断统信UOS系统是否已激活

## 激活状态查询

1. 单击右下角托盘上的 ![authorize1](./fig/authorize1.svg)进入界面。
2. 打开 **控制中心**，选择 **系统信息**，单击 **关于本机**。
3. 查看版本授权栏，查看系统的激活状态（例如：未激活、过期未激活、试用期激活、已激活）。

> ![notes](./fig/notes.svg)说明：如果系统未激活，授权管理图标 ![authorize1](./fig/authorize1.svg)一直常驻在右下角托盘。
